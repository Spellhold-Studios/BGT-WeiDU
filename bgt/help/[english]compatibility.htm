CURRENT BGT-WeiDU COMPATIBILITY LIST

Update: 28 October 2011

This compatibility list is not frequently updated. It is highly recommended that you go to the Mega Mod help forum (http://www.shsforums.net/forum/193-mega-mod-help/) or browse the Big World installation PDF file (http://www.shsforums.net/index.php?app=downloads&showfile=534) for up-to-date compatibility information.

 

Installation tips
-if you want to play a mega-modification containing BGT-WeiDU, it is highly recommended that you investigate the Big World Project, a large collaborative project aiming to maximise the number of modifications installed at the same time. Please go to the Mega Mod Help forum (http://www.shsforums.net/forum/193-mega-mod-help/) and read the pinned topics there in order to get started. Also, consider browsing the Big World installation PDF file (http://www.shsforums.net/index.php?app=downloads&showcat=72).

-if you would rather go about installing a mega-modification alone, install the other modifications first (The Darkest Day, Shadows Over Soubar, Tortured Souls, Check the Bodies, Region of Terror, Never Ending Journey, The Big Picture), then BGT-WeiDU, but you can actually install all the related modifications in any order
-if you want to play the mega-modification with BGT-WeiDU and other modifications, follow the tip above, and then install other modifications on top
-if you want to play BGT-WeiDU without the mega-modification but with other modifications, install BGT-WeiDU first before installing other modifications on top (unless otherwise stated)

Notes on compatibility
-current compatibility between BGT-WeiDU and other modifications are listed below
-note that versions of modifications below those stated may be incompatible
-the compatibility list does not imply that the other modifications mentioned are compatible with each other, e.g. if BGT-WeiDU is compatible with modification A and modification B, then that does not mean that modification A is compatible with modification B. Make sure you understand the specific requirements of each modification by looking at their readmes before you go into hardcore mixing and matching.
-all modifications that add areas to the worldmap will NOT be compatible with BGT-WeiDU UNLESS Revised BP-BGT Worldmap is used
-modifications that affect only the Shadows of Amn and/or the Throne of Bhaal part of Baldur's Gate are best installed just before the transition to Shadows of Amn takes place, although not mandatory
-those modifications that are compatible are assumed to be installed after BGT-WeiDU, unless explicitly stated otherwise (see above and below)
-tentative assignment means I have checked and it seems to be however compatible as I have listed it, but this assignment may not be correct

BGT-WeiDU is partially compatible with:
- Ascension 1.42.1 or above

  • Needs to be installed before BGT-WeiDU if 'Tougher Illasera' component is used, since AR4000.BCS is replaced, resulting in all the BG1 dream cutscenes occurring at rest in a Throne of Bhaal game

- BG2 Tweak Pack v3 or above

  • DO NOT install Components #3140 or #3141 - Faster Chapter 1&2 Cut-Scenes and Dreams; destroys NEWGAME.BCS

- Baldur's Gate Trilogy-Never Ending Journey 2 (BGT-NeJ2) Compatibility Modification v1.2 or below

  • DO NOT install Component #0 - Continuous and Custom Kit Imoen: v1.1; prevents transition from Baldur's Gate to Shadows of Amn from working correctly

- Banter Pack v6 or below

  • Inappropriate Shadows of Amn banters occur during Baldur's Gate and Tales of the Sword Coast

- BGII Improvements v1.8 or below

  • DO NOT install Component #14 - Grandpa's Shield Quest; overwrites PALIN.CRE, a Tales of the Sword Coast file

- Chloe NPC v1.2 or above

  • Overwrites CHLOE.CLG in BGT-WeiDU, so install just before transition

- Druidic Sorcerer Kit v1.7 or above (experience point gain is an inherent part of the modification)

  • Interrupts item shattering of Scimitar and Spear if the 'Option of Installing the GAME-HOSING-ITEM-RESTRICTIONS-USING-DAMN-SCRIPTS (I recommend you don't)' component is installed

- Imoen Romance v1.20 or above

  • Install ONLY just before reaching Irenicus' Dungeon, or use BG2 game only - destroys Imoen's joining/leaving dialogue (goes to Copper Coronet) and causes problems when forcing dialogue with Imoen during Baldur's Gate and Tales of the Sword Coast; Irenicus' dungeon-specific banters do not work

- Planar Sphere Mod

  • Centeol animation replaced by PS:T Lady of Pain animation

- The MTS Trilogy of Crap Part 1 v2 or above

  • DO NOT install Component #0 - Install Keep All BG1 Items On Imported Characters by Kevin Dorner; causes issues with a cutscene hang in Irenicus’ Dungeon

- Viconia Friendship v1

  • Banters may occur during Baldur’s Gate and Tales of the Sword Coast
